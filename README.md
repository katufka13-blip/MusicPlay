Музыкальный сервис🎵

Разработка приложения с использованием Next.js, TypeScript, React, Redux (для управления состоянием), Axios (для HTTP-запросов), HTML, CSS.
Возможности данного сервиса: регистрация; возможность войти\выйти; витрина с доступными треками; встроенный в приложении плеер (прослушивание\пауза, перемотка, повтор трека, случайное воспроизведение); добавление трека в избранное; список подборок.
